    App requirements

    An internet connection is required to install and reset the app, as well as receive any future updates.

    In order for data to persist the app uses local storage which requires IOS/Safari settings, 'Block All Cookies' to be disabled. Uncheck this setting if the app doesn't appear to be functioning correctly.

    For the best iPhone App experience, open the app in IOS Safari and press the icon which looks like an 'up arrow in a box', situated in the bottom tool bar of the Safari Browser. Then select 'Add to Home Screen', followed by 'Add'. An icon will now appear on your phone home screen and the app will launch and function similar to any other apps you have installed. After successfully installing the App to your Home Screen, you no longer need to access the app via the Safari Browser with the URL you originally visited.
